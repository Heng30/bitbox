#### Introduction
It is a easily used bitcoin wallet that based on `Slint-UI`.

#### Features
- [ ] address generation
- [ ] show history transations
- [ ] address book
- [ ] configure bitcoin network
- [ ] send and receive bitcoin

#### How to build?
- Install `Rust` and `Cargo`
- Run `make build`
- Refer to [Makefile](./Makefile) for more information
